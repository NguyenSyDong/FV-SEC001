# FV-SEC001
Thử thách Kỹ sư Phần mềm — Trình tổng hợp hiệu suất quảng cáo

Clone project vể máy

Thêm file ad_data.csv.zip vào thư mục \src\dataFiles (file quá lớn nên em không đẩy lên git được)

Mở bằng IDE và chạy main

Kết quả 2 files sẽ được tạo ở trong thư mục results

Dự án sử dụng:
Java Standard Library
java.io.BufferedReader
java.io.BufferedWriter
java.util.HashMap
java.util.Map

kết quả đầu ra nằm trong thư mục results
